# chunk

지정된 사이즈 길이만큼 배열을 나눈다.


```
chunk(a,b)
  	
 a = 배열
 b = chunk 길이
```

ex)
```
 chunk([1,2,3,4],2)  ==> [[1,2],[3,4]]
 chunk([1,2,3,4],3)  ==> [[1,2,3],[4]]
 chunk([1,2,3,4],5)  ==> [[1,2,3,4]]
```

slice를 이용하여 chunk를 구해보았다.

slice는 기존 배열을 해치지 않는다.

```
 slice(i,i+ chunk사이즈)
 for문을 이용하여 i 값을 chunk사이즈 만큼 올린다.
```
